### Description

Vue 3 Video Player

### Installation
1. Add Video to 'assets' folder
2. Backend (Root Folder) - Default Port 3000
```bash
yarn
```
```bash
yarn dev
```
3. FrontEnd (Client) - Port Random
```bash
yarn
```
```bash
yarn dev
```
